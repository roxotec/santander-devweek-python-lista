# Projeto ETL em Python (Sem API)

Este projeto simula um fluxo ETL simples:

- Extração: leitura de arquivo CSV
- Transformação: geração de mensagens personalizadas
- Carregamento: gravação em novo CSV

## Como executar

1. Certifique-se de ter Python instalado
2. Execute:

python main.py

## Alternativas implementadas

- Uso de CSV completo (sem dependência de API)
